BaRatinAGE est une application Java. Pour l'éxécuter, il faut qu'une machine virtuelle Java soit installée sur votre ordinateur.
Sur l'immense majorité des ordinateurs, cela devrait déjà être le cas.
Il vous suffit alors de double-cliquer sur le fichier BaRatinAGE.jar pour lancer l'application.
Si l'application ne se lance pas, vous pouvez télécharger ou mettre à jour Java à cette adresse: https://www.java.com/fr/download/

Pour accéder à l'aide: help/fr/index.html

Multilinguisme: les contributions pour traduire BaRatinAGE vers d'autres langues sont bienvenues! 
Les fichiers à traduire sont les suivants:
1. Le fichier lang/dico.txt. Ajoutez simplement une colonne contenant la traduction de tous les termes dans la nouvelle langue.
   N'utilisez ni virgules ni points-virgules dans les traductions, car ces caractères sont utilisés comme séparateurs.
2. Tous les fichiers .html dans le dossier "help". Aucune connaissance du langage HTML n'est nécessaire: 
   ouvrez simplement le fichier avec un éditeur de texte (par exemple Notepad++) et traduisez le texte entre les balises HTML
   (<balise> texte à traduire </balise>). Ne modifiez pas les balises elles-mêmes!
Vous pouvez envoyer vos contributions ou poser vos questions en envoyant un mail à baratin.dev@lists.irstea.fr

BaRatinAGE is a Java application. A Java virtual machine is therefore needed to run it.
On most computers, a Java virtual machine should already be installed.
In this case, all you have to do to launch the application is to double-click on the file BaRatinAGE.jar.
If this does not succeed, you can download or upgrade your Java installation here: https://www.java.com/en/download/

To access help: help/en/index.html

The default language when BaRatinAGE is launched first is French.
You can change the language via the Options...Language menu.

Multi-language support: contributions to translate BaRatinAGE are welcome!
The files needing translation are the following:
1. File lang/dico.txt. Just add a column including the translation of all terms in the new language.
   Please do not use commas or semicolons in your translation, since these characters are used as separators.
2. All .html files in the folder "help". No knowledge of the HTML language is required: 
   simply open the file with a text editor (e.g. Notepad++) and translate all text located within HTML tags
   (<tag> text needing translation </tag>). Please do not modify the tags themselves!
Please send your contributions or ask any question by sending an email to baratin.dev@lists.irstea.fr
