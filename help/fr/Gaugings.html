Création d'un jeu de jaugeages

Vous pouvez créer un nouveau jeu de jaugeages de plusieurs façons :

  • via le menu Jaugeages...Ajouter;
  • via le raccourci Ctrl+J;
  • en effectuant un clic droit sur le noeud "Jaugeages" dans l'arborescence de l'explorateur;
  • en cliquant sur le bouton dans la barre d'outils.

Il vous sera demandé de nommer ce nouveau jeu de jaugeages, et vous pouvez en saisir une description. Il ne vous reste plus qu'à importer le fichier de jaugeages en cliquant sur le bouton "Parcourir" et en sélectionnant le fichier de votre choix. A titre d'exemple, vous pouvez sélectionner le fichier "Sauze_gaugings.csv" dans le répertoire "exemple", qui contient 38 jaugeages pour l'Ardèche à Sauze. Le tableau des jaugeages contient les colonnes suivantes :

  • Hauteur exprimée en mètres;
  • Incertitude étendue sur la hauteur jaugée. Dans la plupart des cas nous conseillons de mettre une valeur nulle (voir ce document pour une discussion plus approfondie);
  • Débit exprimé en mètres cubes par seconde;
  • Incertitude étendue sur le débit jaugé, exprimée en pourcentage du débit;
  • Statut du jaugeage : si la case est décochée, le jaugeage ne sera pas utilisé pour l'estimation de la courbe de tarage.

Formats de fichiers

Lorsque vous importez un fichier de jaugeages, vous avez le choix entre deux formats de fichier: le format .csv et le format .BAD.

  • Format .csv : il s'agit d'un format avec séparateur point-virgule (vous pouvez ouvrir le fichier "Sauze_gaugings.csv" dans votre éditeur de texte pour observer le format). Le contenu du fichier est exactement ce que vous voyez dans le tableau de BaRatinAGE représenté ci-dessus (dans la dernière colonne, les cases à cocher sont remplacées par des 0/1). La simplicité de ce format fait que vous pouvez aisément créer/manipuler des fichiers de jaugeages avec votre tableur favori.
  • Format .BAD : ce format est conservé dans un souci de compatibilité avec le logiciel Barème et avec les anciennes versions de BaRatinAGE. En effet vous pouvez exporter depuis Barème un jeu de jaugeage au format .BAD. Cependant, nous déconseillons de manipuler directement ce format de fichier avec votre tableur, car le contenu du fichier n'est pas le même que ce que vous voyez dans le tableau de BaRatinAGE. Pour être plus précis : les incertitudes y sont toutes exprimées en écart-type (et non en incertitude étendue) et en absolu (et non en % pour le débit).

Remarques sur la gestion des jaugeages dans BaRatinAGE

A l'heure actuelle, la gestion des jaugeages est rudimentaire dans BaRatinAGE : la seule modification possible est l'activation/désactivation d'un jaugeage. Si vous souhaitez modifier une valeur ou ajouter un jaugeage, vous devez modifier le fichier source. La raison est qu'il existe déjà des logiciels spécialisés dans la gestion des jaugeages (Barème en particulier), et qu'il nous semble préférable de favoriser la compatibilité avec ces logiciels plutôt que de ré-inventer la roue. Néanmoins, nous implémenterons quelques outils simples dans les futures versions de BaRatinAGE (saisie et modification des jaugeages, inclusion des dates et des méthodes de jaugeages, filtres élémentaires).
