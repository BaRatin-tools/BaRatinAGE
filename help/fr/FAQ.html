BaRatinAGE

Foire aux questions

Note: cette section sera mise à jour au fil des questions que nous recevrons.

Installation

Question : je travaille sous Linux, puis-je utiliser BaRatinAGE ?

Réponse : oui, depuis la version 2-2, BaRatinAGE peut être utilisé sous Windows et Linux !

---

Question : je travaille sous Mac, puis-je utiliser BaRatinAGE ?

Réponse : pas pour le moment. Si vous le souhaitez, nous pouvons néanmoins vous accompagner pour tenter de compiler une version Mac du logiciel. Dans ce cas, contactez-nous!

---

Formation

Question : existe-t-il des formations à BaRatinAGE ?

Réponse : oui, plusieurs options sont à votre disposition :

  • Utiliser les supports de formation en français, anglais et espagnol disponibles ici;
  • Suivre la formation courbe de tarage de l'IFORE ;
  • Suivre la formation Statistiques avancées pour le calcul d'incertitude de l'IFORE ;
  • Nous pouvons organiser à la demande des sessions dédiées à BaRatinAGE, typiquement sur 2 jours. Pour plus d'informations sur ces "baratinades", contactez-nous !

---

Licence, codes sources, etc.

Question : puis-je distribuer BaRatinAGE à tous mes collègues, ou sur mon site web ?

Réponse : depuis la version 2-2, oui. Cependant, nous recommandons à chaque utilisateur de nous envoyer un courriel pour demander l'inscription à la liste de diffusion des utilisateurs de BaRatin. Ceci permet de recevoir des informations sur les nouvelles versions du logiciel.

---

Question : puis-je avoir accès au code source de BaRatinAGE ?

Réponse : BaRatinAGE et BaRatin sont maintenant open source (licence GPL3) et disponibles sur GitHub. Les futures versions du logiciel utiliseront le code de calcul BaM!, lui aussi disponible en open source (licence GPL3).
