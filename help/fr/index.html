Généralités

BaRatinAGE est un logiciel permettant d'estimer des courbes de tarage ainsi que leurs incertitudes, et de propager ces incertitudes aux hydrogrammes. La méthode est basée sur une analyse hydraulique de la station hydrométrique et sur l'estimation bayésienne de la courbe de tarage. BaRatinAGE est l'acronyme de BAyesian RATINg curve Advanced Graphical Environment.

Pour installer BaRatinAGE, il vous suffit d'extraire le fichier zip que vous avez dû télécharger ici dans le dossier de votre choix, pourvu que vous ayez les droits d'écriture dans ce dossier. Double-cliquez sur le fichier BaRatinAGE.jar, et (si tout va bien...) c'est parti!

Note 1: pour l'instant, BaRatinAGE ne fonctionne que sous Windows. Une version Linux devrait être disponible sous peu.

Note 2: BaRatinAGE étant une application Java, elle réclame qu'une machine virtuelle Java soit installée sur votre ordinateur. Sur l'immense majorité des ordinateurs, cela devrait déjà être le cas. Dans le cas contraire, vous pouvez télécharger ou mettre à jour Java ici.

Note 3: la langue par défaut au premier lancement de BaRatinAGE est le français. Vous pouvez changer la langue via le menu Options...Langue

Principaux objets dans BaRatinAGE

BaRatinAGE manipule les objets suivants, dont le fonctionnement sera détaillé dans les pages qui suivent:

  • La configuration hydraulique est la définition de l'ensemble des contrôles qui gouvernent la relation physique hauteur-débit, et de leur succession/combinaison lorsque la hauteur d'eau augmente. La configuration hydraulique induit l'équation qui sera utilisée pour la courbe de tarage.
  • Le jeu de jaugeages est l'ensemble des mesures ponctuelles de hauteur et de débit qui seront utilisées pour estimer les paramètres de la courbe de tarage.
  • Le modèle d'erreur restante définit les propriétés de l'erreur structurelle de la courbe de tarage.
  • Le limnigramme est la série temporelle de hauteur mesurée en continu au droit de la station hydrométrique.
  • La courbe de tarage est la relation hauteur-débit estimée à partir d'une configuration hydraulique, d'un jeu de jaugeages et d'un modèle d'erreur restante.
  • L'hydrogramme est la série temporelle de débit calculée à partir d'un limnigramme et d'une courbe de tarage.

Les quatre premiers objets dans la liste ci-dessus sont des données d'entrée pour les calculs effectués par BaRatinAGE, et doivent donc être spécifiés par l'utilisateur. A l'opposé, les objets courbe de tarage et hydrogramme représentent les résultats des calculs effectués par BaRatinAGE. Ils pourront être exportés sous différents formats.
