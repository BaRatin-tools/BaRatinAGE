Frequently asked questions

Note: This section will be updated with the questions we receive.

Installation

Question: I'm working on Linux, can I use BaRatinAGE?

Answer: yes, from version 2-2, BaRatinAGE can be used on Windows and Linux OS!

---

Question: I'm working on Mac, can I use BaRatinAGE?

Answer: not yet. However, if you're courageous enough to consider compiling the software on a Mac, we'll be happy to assist. Please get in touch in this case!

---

Training

Question: are there training sessions on BaRatinAGE?

Answer: yes, several options are available. Training material in English, French and Spanish is available. We can organise on request sessions dedicated to BaRatinAGE, typically during 2 days. For more information on these "baratinades" contact us!

---

License, source codes, etc.

Question: can I distribute BaRatinAGE to all my colleagues, or on my website?

Answer: From version 2-2, yes. However, we recommend each user to send us an email to suscribe to the BaRatin users mailing-list in oder to receive information on new versions.

---

Question: can I have access to the source code of BaRatinAGE?

Answer: Yes, BaRatinAGE, BaRatin are now open source (license GPL3) and available on GitHub. Future versions will use the computing code BaM!, also available with an open source GPL3 licence.
