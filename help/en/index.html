BaRatinAGE

Overview

Introduction

BaRatinAGE is a software for estimating rating curves and their uncertainties, and propagate these uncertainties to flow/discharge series. The method is based on the hydraulic analysis of the hydrometric station and on the Bayesian estimation of the rating curve. BaRatinAGE stands for BAyesian RATINg curve Advanced Graphical Environment.

To install BaRatinAGE, simply extract the zip file you downloaded here into the folder of your choice, provided you have write access to this folder. Double-click the BaRatinAGE.jar file and (hopefully...) it should be ready to go!

Note 1: BaRatinAGE being a Java application, it requires a Java Virtual Machine installed on your computer. On the vast majority of computers, it should already be the case. Otherwise, you can download or update Java here.

Note 2: On Windows, the association between Java and .jar files is sometimes broken. You may try to fix this association (some hints here), or alternatively you may use this file to start BaRatinAGE (open it with a text editor, modify the path to Java, save/close then double-click on the file).

Note 3: the default language when BaRatinAGE is launched first is French. You can change the language via the Options...Language menu.

Main objects in BaRatinAGE

BaRatinAGE manipulates the following objects (which will be further detailed in the following pages):

  • The hydraulic configuration is the definition of all the controls that govern the physical stage-discharge relationship, and their succession/combination when the water level rises. The hydraulic configuration induces the equation of the rating curve.
  • The set of gaugings is the set of discrete measurements of stage and discharge that will be used to estimate the parameters of the rating curve.
  • The remnant error model defines the properties of the structural error of the rating curve.
  • The stage series is the time series of water levels measured continuously at the hydrometric station.
  • The rating curve is the stage-discharge relationship estimated from a hydraulic configuration, a set of gaugings and a remnant error model.
  • The flow series is the time series of discharge calculated from a stage series and a rating curve.

The first four items in the above list are input data for the computations done by BaRatinAGE, and must therefore be specified by the user. In contrast, objects rating curve and flow series represent the results of BaRatinAGE computations. They can be exported in various formats.
